# Javascript-juego-culebrita
Sintaxis básica clásico juego serpiente

Game snake

El juego de la serpiente (snake) es un clásico de los videojuegos que consiste en controlar una serpiente que se mueve por una cuadrícula y debe comer manzanas para crecer y aumentar su velocidad. El juego termina cuando la serpiente choca con los bordes de la cuadrícula o consigo misma. Este juego es muy útil para enseñar conceptos básicos de programación en JavaScript, como:

- Variables: se utilizan para almacenar valores que cambian durante el juego, como la posición, la dirección y el tamaño de la serpiente, el puntaje, el intervalo de tiempo, etc.
- Arrays: se utilizan para representar la serpiente como una colección de segmentos que tienen una coordenada x y una coordenada y. También se pueden usar para crear la cuadrícula como una matriz bidimensional de celdas.
- Funciones: se utilizan para definir acciones que se ejecutan en el juego, como dibujar la serpiente y la manzana, mover la serpiente, detectar colisiones, generar números aleatorios, actualizar el puntaje, etc.
- Eventos: se utilizan para capturar las entradas del usuario, como las teclas presionadas o los botones tocados, y cambiar la dirección de la serpiente según el evento.
- Bucles: se utilizan para recorrer los elementos del array de la serpiente y actualizar su posición y apariencia. También se pueden usar para crear y dibujar la cuadrícula.
- Condicionales: se utilizan para verificar si se cumplen ciertas condiciones en el juego, como si la serpiente ha comido una manzana, si ha chocado con los bordes o consigo misma, si el juego ha terminado, etc.

El juego de la serpiente es un ejemplo sencillo pero divertido de cómo usar JavaScript para crear interactividad y dinamismo en una página web.
